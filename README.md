01.10_Java语言基础(标标识识符符中中常常见见的的命命名名规规则则)(了解)
	见名知意
	A:包
		最好是域名倒过来,要求所有的字母小写
		
	B:类或者接口
		如果是一个单词首字母大写
		如果是多个单词每个单词首字母大写(驼峰标识)
		
	C:方法或者变量
		如果是一个单词全部小写
		如果是多个单词,从第二个单词首字母大写
		
	D:常量
		如果是一个单词,所有字母大写
		如果是多个单词,所有的单词大写,用下划线区分每个单词
